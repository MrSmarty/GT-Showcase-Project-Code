Hexadecimal

Hexadecimal is a base 16 numbering system used in computer science. Base 16 means that there are 16 digits for each place value starting at 0 and ending at F (15). To convert a Hexadecimal number to a decimal, or "normal", number is really quite simple. Let's use 7B2A.

  1. Take the first digit (A), convert it to a normal number (10) and multiply it by 160 (or 1) to get 10.
  2. Take the second digit (2), and multiply it by 161 to get 32.
  3. Convert the third digit (B) to 11 and multiply by 162 to get 2,816.
  4. Multiply the last digit (7) by 163 to get 28,672.
  5. Add up the 4 numbers and that is your number (31,530).
We use Hexadecimal because we can store larger numbers in smaller space while still lining up with Binary.
